Exercise 4 b

Opdracht: Verander de code en het lampje, zodat er iets anders gebeurt.

Ik heb het lampje vervangen voor een Servomotor. In de code van Arduino heb ik veranderd dat niet het lampje aan moet gaan, maar dat de Servometer moet gaan draaien. Dit gebeurt wanneer ik bij processing met mijn muis op het blokje kom. Wanneer ik met mijn muis van het blokje afgaat, draait de Servomoter terug. Je ziet de Servomotor heen en weer gaan, omdat ik met de zelfde hand de muis moet bewegen.
